# Smart Permit Management System (SPMS)

Smart Permit Management System (SPMS) is a role-based web application designed to manage permit applications, approvals, and tracking across multiple user roles including Citizens, Staff, and Admins.

The system simulates a real-world workflow where applications are submitted, reviewed, and processed through structured approval stages.

---

## Overview

SPMS is built to streamline the permit lifecycle:

- Citizens submit applications with supporting documents  
- Staff review and take action on applications  
- Admins manage users and oversee the system  

The platform ensures transparency, traceability, and controlled access across all roles.

## Key Features

### Authentication & Authorization
- Secure login and registration  
- Role-based access control for Citizens, Staff, and Admins  

### Citizen Module
- Submit permit applications  
- Upload required documents  
- Track application status  

### Staff Module
- Review submitted applications  
- Approve, reject, or request additional information  
- Add comments and feedback  

### Admin Module
- Manage system users  
- Oversee all applications  
- Control workflow processes
